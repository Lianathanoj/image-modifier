Image Modification Using Rule of Thirds

Alex Le, Maggie Shi, Eric Yan, Victor Chen, Jeffrey Ma, Jonathan Lian

Problem Statement

The goal of the project is to process an image to be more pleasing to the eye according to the rules of third, a photographic guideline used in visual compositions. This will be carried out and compared through different methods. The expected input is an image with a person or group of people, and the expected output is an image with points of interest repositioned to be at "pleasing" points according to the rule of thirds.

Technical Approach

We will have multiple methods for implementing rule of thirds, which can be divided into three different parts. For these three parts (pre-processing, feature selection, and warping), we have multiple, different methods for each that will be mixed-and-matched.

Pre-Processing Methods
  • Smoothing
    • Gaussian filter
    • Box filter
  • Median Filter
  • Data Pre-Processing for Semantic Segmentation
    • Augmentation
    • Balancing
    • Noise
Selecting Points of Interest
  • Method 1 (Energy function + Seam Carving)
    1. Find the points of interest of the image according to high energy points and places of high energy, where there are the most changes
    2. Ensure interest points are sufficient distances away from each other
  • Method 2 (Feature Selection)
    1. Create a classifier to detect important features (e.g. human outline) that will be used in determining which portion of an image to move/warp
  • Method 3 (Semantic Segmentation)
    1. Apply segmentation techniques to determine human outline
    2. Model - Mask R-CNN
Image Modification
  • Homography Transformation + Seam Carving
    1. Move the interest points to those areas using homography transformation techniques
    2. Use seam carving technique to resize those images to a standard, rectangular shape

Experiment and Results

  1. Experiment Setup
    1. We will have people compare the outputs of each method for the same image sets, to see how well each method does
    2. Our population will be a sample of at least 50 people from the Georgia Tech student body
    3. The original set of images will comprise the control set
    4. The image method that generates the most pleasing output out of the three, according to our sample set would be the “best” method
  2. What datasets we will use
    1. Coco (public dataset)
    2. PasVal
  3. What we will implement
    1. We will implement the classifier in Python using libraries such as scikit-learn
    2. We will also use OpenCV functions to construct the energy function, seam carving, and homography transformations
    3. We will feed image inputs to Mask R-CNN and use their outputs to determine segmentation
  4. Success Definer
    1. We will poll Georgia Tech students to determine which method is the best (as in which method’s output people prefer for a given image)
  5. List of experiments to be performed
    1. Energy function + seam carving and Homography transformation + seam carving
    2. Feature Selection and Homography transformation + seam carving
    3. Segmentation and Homography transformation + seam carving
