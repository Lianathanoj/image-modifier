Image Modification Using Rule of Thirds

Alex Le, Maggie Shi, Eric Yan, Victor Chen, Jeffrey Ma, Jonathan Lian

Abstract

The rule of thirds is an image metric used by photographers to increase the overall aesthetic of a composition, wherein an image is overlaid onto a 3 x 3 grid and the subject matter becomes more qualitatively agreeable if it falls within the middle intersections of the grid.

Introduction

The goal of the project is to thus process an image to be more pleasing to the eye according to the rule of thirds. The expected input is an image with a person or group of people, and the expected output is an image with points of interest repositioned to be at "pleasing" points according to the rule of thirds.

Technical Approach

Our technical approach can be subdivided into multiple steps. First, we will be utilizing a Mask R-CNN which has been pre-trained upon a large set of images. We will then be able to extract an image's important areas of interest which will allow us to later transform the image to adhere to the rule of thirds.

1) Determining Bounding Box and Masks of Points of Interest Via Feature Extraction
Using a Mask Regional Convolutional Neural Network (R-CNN) from https://github.com/matterport/Mask_RCNN, we extracted areas of interest such as the relevant bounding boxes and semantic segmentation masks for people that were at different points in a picture. Note that the Mask R-CNN already had a pre-trained classifier that we opted to use because it sufficiently was able to detect different classes. We tested self-retrieved pictures to make sure that humans were being correctly detected and to ensure that our bounding box correctly encapsulated human outlines.

    4. The image method that generates the most pleasing output out of the three, according to our sample set would be the “best” method
